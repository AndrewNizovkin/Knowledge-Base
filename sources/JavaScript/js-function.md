# Функции

Для создания функции существуют разные синтаксисы:

- **Function Declaration** (Объявление Функции):

```js
function имя(параметры) {
    // тело функции
}
```
> Название функции должно обозначать действие. В JS используется `camelCase`

Функция может возвращать значение или не возвращать.

```js
function showMessage(from, text) {
    console.log(`${from}: ${text}`);
}

function getSum(a, b) {
    return a + b;
}
```

- **Function Expression** (Функциональное Выражение). Данный синтаксис позволяет нам создавать новую функцию в середине любого выражения.

```js
let sayHi = function() {
  alert( "Привет" );
};
```
